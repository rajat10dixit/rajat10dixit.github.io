HTML5 Document Structure

An HTML5 document typically consists of several key elements, each serving a specific purpose. Below is an explanation of these elements, including meta tags for a favicon and keywords.

DOCTYPE Declaration

The declaration defines the document as an HTML5 document. It helps the browser to render the page correctly.

HTML Tag

The tag is the root element of an HTML document. The lang attribute specifies the language of the document.

Head Section

The section contains meta-information about the document, such as its title, character set, viewport settings, and links to stylesheets or scripts.

Charset Meta Tag

The tag specifies the character encoding for the document, ensuring that it is displayed correctly across different devices and browsers.

Viewport Meta Tag

The tag controls the layout on mobile browsers by setting the width and scale of the viewport.
